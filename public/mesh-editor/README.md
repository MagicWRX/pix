# Mesh Potato Editor

A web-based 3D mesh editor built with Three.js.

**Live Demo:** https://experiments.obsoleterobot.com/mesh-editor/

## Features
- Grid planes (XZ/XY/YZ) with toggle (keys 1/2/3)
- Object selection and transform controls
- Snapping: Shift (axis), Alt (grid), Alt+Shift (fine)
- Save/Load JSON, Export GLB
- Theme-driven UI/UX

## Keyboard Shortcuts
| Key | Action |
|-----|--------|
| 1/2/3 | Grid plane XZ/XY/YZ |
| G | Toggle grid snap |
| T/R/S | Translate/Rotate/Scale mode |
| Delete | Delete selected |
| Ctrl+D | Duplicate |
| Ctrl+S | Save |
| Ctrl+E | Export GLB |
